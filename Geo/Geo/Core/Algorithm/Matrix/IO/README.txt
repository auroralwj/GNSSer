定义：
文档按区存储，分区以特定的字符进行区分。
二进制分区以特定字符分割。

文件分为头部（文档说明区）和内容区：

头部区
一般包含文档创建时间，文档所有者、文档大小、注释等信息。
第1个为byte，表示矩阵文件的版本，对应数字 1-255
第2个为byte，表示矩阵的类型，对应C#的枚举类型
第3、4个数为int32，分别表示 矩阵的行和列。
第5个为内容元素的数量
第6个为8个Char字符的矩阵名称或标识
第7个为long表示的时间
第8个为8个Char字符的创建者名称
之后为Char注释,直到遇到结尾字符为止。



内容区：
第1个为1个Char的内容开始标识符
第2个开始为浮点数（双精度、单精度或整型）记录
内容去以内容结尾标识符结束
